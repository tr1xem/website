वीडियो के साथ-साथ चलें और यहां से कमांड कॉपी करें

> [!NOTE]
> सभी पैकेज नाम आर्क लिनक्स के लिए हैं, इसलिए किसी अन्य डिस्ट्रो के लिए आपको यह खोजने के लिए गूगल की आवश्यकता हो सकती है कि यह क्या है

### चरण 1

#### वायरशार्क या वायरशार्क-क्यूटी इंस्टॉल करें

इसे अपने डिस्ट्रो के अनुसार करें, फिर

**एक बार की प्रक्रिया**

- खुद को वायरशार्क ग्रुप में जोड़ें - (**महत्वपूर्ण**)

```bash
sudo groupadd wireshark
sudo usermod -a -G wireshark $USER
```

### चरण 2

#### यूएसबीमोन इंस्टॉल करें

इसे अपने डिस्ट्रो के अनुसार करें, फिर

**एक बार की प्रक्रिया**

- खुद को यूएसबीमोन ग्रुप में जोड़ें - (**महत्वपूर्ण**)

```bash
sudo groupadd usbmon
sudo usermod -a -G usbmon $USER
```

फिर इसे चलाएं: (**महत्वपूर्ण**)

यह आपको रूट अनुमतियों के बिना यूएसबी पैकेट कैप्चर करने देगा

```bash
sudo tee /etc/udev/rules.d/99-usbmon.rules <<'EOF'
SUBSYSTEM=="usbmon", GROUP="wireshark", MODE="0640"
EOF
sudo udevadm control --reload-rules
sudo udevadm trigger
```

> [!NOTE]
> आपको लॉग आउट करके फिर से लॉग इन करने की आवश्यकता हो सकती है

### चरण 3

#### यूएसबीमोन मॉड्यूल लोड करें

चलाएं

```plain
sudo modprobe usbmon
```

यह कर्नेल मॉड्यूल लोड करेगा

## चरण 4 - 6

- यूएसबी पैकेट कैप्चर करें
- आपको जो करने के लिए कहा गया है वह करें
- फ़ाइल का नाम दें और भेजें।

वाह, आपने अपने यूएसबी पैकेट कैप्चर कर लिए हैं 🫡
