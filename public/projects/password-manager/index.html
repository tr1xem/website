A secure, minimalist command-line password manager built on top of the Unix philosophy. Uses GPG encryption and Git for synchronization across devices.

Features⌗

  • GPG Encryption - Military-grade encryption for all password data
  • Git Synchronization - Sync passwords across multiple devices
  • Cross-platform - Works on Linux, macOS, and Windows (WSL)
  • Import/Export - Support for various password manager formats
  • Dmenu Integration - Quick password selection with dmenu
  • Android Support - Compatible with Android Password Store app

Implementation⌗

Built using standard Unix tools and following the suckless philosophy of doing one thing well. The password store is a simple directory tree with each password as an encrypted file.

Tech Stack⌗

  • Bash - Core scripting and CLI interface
  • GPG - Encryption and key management
  • Git - Version control and synchronization
  • Pass - Standard Unix password manager foundation

Security Features⌗

  • Zero-knowledge architecture
  • Offline-first design
  • Open source and auditable
  • No network dependencies for core functionality
